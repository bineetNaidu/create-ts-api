# create-ts-api

A utility command to set up TypeScript, NodeJS, Express/GraphQL API project.

## Installation

Install create-ts-api with npm

```bash
  npm i -g create-ts-api
  create-ts-api
  cd <your-project-name>
  # OR
  npx create-ts-api
  cd <your-project-name>
```
